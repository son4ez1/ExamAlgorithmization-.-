Данные в компьютере организованы в виде битов и байтов. Чем больше файл по размеру, тем большая последовательность нулей и единиц необходима для его описания. Бит - это наименьшая единица информации, которая может принимать два значения: 0 или 1. Байт - это группа из 8 битов, которая может представлять целые числа от 0 до 255.
Типы данных:
 integer - это целые числа
 float - это числа с плавающей точкой
 char - это символы, которые могут быть буквами, цифрами, знаками препинания и другими символами.
boolean - это данные, которые могут принимать только два значения: true или false.
string - это строки используются для хранения текстовой информации.
